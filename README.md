# Homework Four

This repository contains the files for Homework Four of the N215 course.

## Structure

- `index.html` - main HTML file
- `css/styles.css` - compiled CSS
- `scss/styles.scss` - Sass source
- `scss/variables.scss` - Sass variables
- `images/` - image assets

## Setup

1. Run `npm install` to install dependencies.
2. Compile Sass using `npm run sass` or your preferred tool.

## Usage

Open `index.html` in a browser to view the project.

## Notes

This project demonstrates basic Sass compilation and styling.
